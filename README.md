# Prime Axis North America Limited — Website

This is a Vite + React + Tailwind site, ready to deploy on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this folder to a GitHub repo (recommended).
2. In Vercel, click **Add New → Project**, import the repo.
3. Vercel should auto-detect **Vite**.
   - Build Command: `npm run build`
   - Output Directory: `dist`
4. Deploy.
